WordPress URL Grabber Settings

Configure how the extension processes WordPress URLs

URL Validation
Control whether the extension validates each URL for accessibility before displaying it. This affects both performance and accuracy of the results.
Performance Impact:
• Enabled: 30-60+ seconds processing time, but filters out broken/private URLs
• Disabled: 2-5 seconds processing time, but may show inaccessible URLs
Results Limit
Set the maximum number of URLs to fetch per post type. Higher values will show more results but may take longer to process.
Note: Some WordPress sites may have thousands of posts. Higher limits will fetch more complete results but will take longer to process, especially with URL validation enabled.
Result Caching
Cache results for faster subsequent loads. Cached data will be refreshed automatically after the specified time period.
Performance Impact:
• Enabled: First load takes full time, subsequent loads are instant
• Disabled: Every load fetches fresh data from the WordPress site
